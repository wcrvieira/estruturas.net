# estruturas.Net
Neste projeto, o objetivo é explorar a aplicação das estruturas condicionais como if-else, switch e as de repetição (While, Do...While e For), com elaboração de operações cotidianas e práticas. 

# Exemplos de aplicação: 
Escolha uma das opções, abaixo, e implemente na interface principal do projeto utilizando o evento Click dos botões.
- Tabuadas variadas
- Cálculos de área, perímetro, raios e outros
- Cálculo de fatoriais
- Geração de mensagens diversas
- Validação de senhas
- Geração de menus de opções
- Exibir números primos até um certo valor
- Calcular o quadrado de 1 até 100
- Exibir a raiz quadrada de números até 400
- Encontrar o maior e o menor de 5 números
- Operações com porcentagens e outros

# Ferramentas:
- IDE Microsoft Visual Studio 2022 ou superior
